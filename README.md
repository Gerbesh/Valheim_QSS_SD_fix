# QuickStackStore - ItemDrawers Compatibility

A lightweight compatibility plugin for Valheim that adds support for ItemDrawers / Storage Drawers to QuickStackStore.

Standalone BepInEx plugin. No source modifications.

## Features
- QuickStack support for ItemDrawers
- Uses native ItemDrawers logic
- Configurable radius and behavior
- Safe, minimal, and network-correct

## Installation
Place the DLL into:
BepInEx/plugins/QuickStackStore.ItemDrawersCompat/
