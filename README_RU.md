# QuickStackStore - ItemDrawers Compatibility

Мод-совместимость для Valheim, добавляющий поддержку ItemDrawers в QuickStackStore.

Отдельный BepInEx-плагин без правок исходных модов.

## Возможности
- Быстрый стак в ItemDrawers
- Использует нативную логику ItemDrawers
- Настраиваемый радиус и параметры

## Установка
Поместите DLL в:
BepInEx/plugins/QuickStackStore.ItemDrawersCompat/
